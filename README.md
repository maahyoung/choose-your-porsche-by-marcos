# Choose Your Porsche by Marcos

**Repository package release:** 27.0

Interactive Porsche 911 GT3 RS configurator concept built with Next.js, React Three Fiber, Three.js and Zustand.

## V27 highlights

- Tap-versus-drag gesture protection for doors and hood.
- Limited cockpit look-around using mouse, touch or trackpad drag.
- Adaptive scene framing for phones, tablets, compact displays, desktops and ultrawide screens.
- Real headlight spotlights plus layered flare/glow.
- Independent amber layers for mirror repeaters and rear indicator tips.
- Mobile DPR and shadow adjustments for more stable performance.

## Run locally

```bash
npm install
npm run dev
```

## Production checks

```bash
npm run lint
npm run build
```
