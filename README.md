# Choose Your Porsche by Marcos

Interactive Porsche 911 GT3 RS configurator concept built with Next.js, React Three Fiber, Three.js and Zustand.

Current packaged release: **v27.1**.

## Run locally

```bash
npm install
npm run dev
```

## Production checks

```bash
npm run lint
npm run build
```

This is an unofficial personal learning project and is not affiliated with Porsche AG.
