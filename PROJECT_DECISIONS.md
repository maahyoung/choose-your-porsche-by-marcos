# Project decisions

## Identity

- Project name: **Choose Your Porsche — by Marcos**
- Personal, unofficial coding-learning project
- Creator mark: cursive **MMC** signature
- Exported images should include the project name and MMC signature
- Fixed European-style plate: **MARCOS911**

## Vehicle target

- Porsche 911 GT3 RS, generation 992
- Exterior first; interior in a later milestone
- Initial configuration: white paint with black details
- Real performance data must be sourced from official Porsche material before it is displayed

## Visual direction

- Premium official-configurator discipline
- Cinematic camera language inspired by modern racing-game presentation
- Dark reflective glass platform
- Predominantly red environmental lighting
- Abstract blurred-track and speed-light atmosphere
- No background video in order to protect performance

## Interaction

- Mouse and touch 360° exploration
- Step-based right-side configurator
- English and Portuguese
- Landscape-first experience across desktop, tablet and phone
- Portrait mobile presents a polished rotate-device screen
- Build summary uses an automatic Forza-inspired camera orbit and detail shots

## Planned configuration categories

1. Paint
2. Wheels and brakes
3. Exterior packages
4. Graphics
5. Aero and DRS
6. Stance
7. Lighting
8. Build summary

## Official vs concept modes

Factory-realistic options should remain visually distinct from experimental options.

Examples of future concept-only controls:

- wing delete
- suspension settings outside a factory-realistic range
- non-factory component combinations

Concept controls should be labeled clearly and should never be presented as factory Porsche configurations.

## Performance strategy

- Performance: modern phone fallback
- Balanced: default for iPad and modern mobile devices
- Ultra: stronger desktops and tablets with safe limits
- No requirement to support old devices
- Reflective materials, shadow resolution and animated-light count scale by quality mode

## Asset policy

No copied commercial 3D model, official Porsche image, video or engine recording should be committed without an appropriate license. Placeholder assets must be clearly identified as placeholders.
